## Usage Rules

1. 새 실험 결과나 일회성 점검 결과는 이 디렉터리에 둔다.
2. planning 우선순위는 이 디렉터리 문서로 직접 결정하지 않는다.
3. active strategy 문서에는 결과 전체를 복붙하지 말고, 필요한 경우 이 디렉터리 문서로 링크만 남긴다.
4. evidence 문서는 날짜 또는 실험명 기준으로 추가하고, 기존 파일에 새 상태를 덮어쓰지 않는다.
